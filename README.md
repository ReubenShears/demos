# Optimally — Demo Landing Pages

Static host for VSL call-funnel demo landing pages, one per prospect.

- **Live domain:** https://demos.optimally.ltd
- **Each demo:** `<slug>/index.html` → served at `https://demos.optimally.ltd/<slug>`
- **Deploys:** auto-built by Vercel (the `demos` project) on every push to `main`.

Pages are generated by the `vsl-funnel-demo` skill (Firecrawl brand scrape → Google Stitch →
CTA partner-tracking → deploy here → Baserow log → Slack post). Do not hand-edit unless you know
what you're doing — re-running the skill overwrites a slug's `index.html`.
