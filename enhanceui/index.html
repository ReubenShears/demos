Teach Me Design — a step-by-step curriculum that builds the foundation you're missing

Not another scattered collection of tips. A proper sequence — from color fundamentals through typography, component design, visual hierarchy and implementation — that compounds chapter by chapter.

Colors

Color theory, the wheel, psychology, contrast ratios and how to build a production-ready color schema you can drop into any project.

Typography

Typefaces, font families, legibility, text hierarchy and building a typography schema that scales consistently across every page.

Component Design

Spacing, grids, buttons, forms, cards, tables, accordions — every UI component category with the principles behind why good ones look good.
